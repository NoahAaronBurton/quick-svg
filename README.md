
# Quick SVG
  
  
## Description
  Enter up to three letters, a shape, and colors to create your own custom SVG file.


  ![placeholder img](./examples/example.svg)
  
  - Purpose of Project:

    This app makes it easy to generate an SVG icon/logo from the command line.
